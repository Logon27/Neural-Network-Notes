### Create venv
```
python -m venv venvNotes
```

### Activate venv
```
. venvNotes/Scripts/activate
```

### Installing dependencies
```
python -m pip install --upgrade pip
pip install -r requirements.txt
```

### Run the site
```bash
# From Neural-Network-Notes directory
mkdocs serve
```
